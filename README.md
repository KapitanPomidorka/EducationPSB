# EducationPSB
Educational platform HackChange2025

Для запуска требуется postgresql (pgAdmin4), Visual Studio Insiders. 
После запуска проекта в браузере вписывается localhost:5000. Есть отображение на данный момент только для курсов - их можно просматривать на главной странице и в отдельном окне с детализацией данных.
